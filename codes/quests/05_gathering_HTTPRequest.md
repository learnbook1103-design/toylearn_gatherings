## prompt1 (캔들 패턴 및 모멘텀 측정 분석)
```
각 거래일의 일중 변동성 지표 (고가 - 저가) / 종가를 계산해 줘. 이 지표가 평균 대비 1.5배 이상 급증한 날짜들을 포착하고, 이는 시장의 '모멘텀 확대' 관점에서 어떤 의미인지 해석해 줘. 또한, 윗꼬리가 길게 달린 날 (윗꼬리 길이가 전체 캔들 길이의 50%를 초과하는 날) 2개를 선정하고, 이는 **'단기적인 차익 실현 압력'**이 최고조에 달했다는 측면에서 투자자에게 어떤 경고 시그널이 될 수 있는지 분석해 줘.
\n\n출력은 정확한 JSON 형식이어야 해. 예시는 아래와 같아:\n\n{\"score\": (0~100), \"reason\": (상세 분석 근거)}\n\n
데이터는 아래와 같아:
{{ $json.data }}
```

## prompt2 (거래량 기반 심리 분석)
```
제시된 주가 데이터에서 **거래량 급증일 (평균 거래량 대비 3배 이상)**을 식별해 줘. 해당 급증일의 종가와 전일 대비 등락률을 함께 제시하고, 이를 근거로 해당 대량 거래가 '강력한 매집' 또는 '차익 실현(분산)' 중 어느 쪽 시그널로 해석되는지 판단해 줘. 특히, 주가 횡보 또는 하락 구간에서 거래량이 급감한 시점을 포착하고, 이는 이후 주가 상승의 '매도 물량 소진' 관점에서 어떤 의미를 가지는지 설명해 줘.
\n\n출력은 정확한 JSON 형식이어야 해. 예시는 아래와 같아:\n\n{\"score\": (0~100), \"reason\": (상세 분석 근거)}\n\n
데이터는 아래와 같아:
{{ $json.data }}
```
